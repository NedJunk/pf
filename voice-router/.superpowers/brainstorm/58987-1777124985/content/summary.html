System Overview

A context-aware voice facilitator for project refinement and personal development.

The "Smart Entry" Workflow
1. CAPTURE

Voice Interaction: Low-latency mobile/on-the-go interface.

The Router: Active facilitator that probes, clarifies, and guides the session.

2. CONTEXT

Dynamic Injection: Router pulls from Project Maps, Knowledge Graphs, and High-Level Goals.

The Whisperer: Async Experts (PM, Architect) feed insights to the Router in real-time.

3. ARTIFACTS

Async Reasoning: Powerful models process the full session transcript.

Output: Updated project specs, task lists, and brain-dump logs saved as Markdown.

We've moved from "Task Tool" to "Development Partner." This architecture focuses on reducing the friction of capture while maximizing the utility of your existing project context.
